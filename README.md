# Valoran Training

VALORANT 감도를 기준으로 60초 에임 루틴을 연습하고, 브라우저에 개인 기록을 저장하는 데스크톱용 3D 에임 트레이너입니다.

## 시작하기

```bash
npm install
npm run dev
```

브라우저에서 표시된 로컬 주소를 열고, 실제 마우스 DPI와 VALORANT 감도를 입력하세요. 프로덕션 번들은 `npm run build`로 생성합니다.

## 지원 환경과 조작

- Windows 데스크톱 Chrome과 물리 마우스를 우선 지원합니다.
- WebGL과 Pointer Lock이 필요합니다. 첫 훈련 시작 시 화면을 클릭해 마우스 잠금을 허용하세요.
- 시야 이동: 마우스 / 사격: 왼쪽 클릭 / 일시정지: `Esc` 또는 포커스 이탈 / 재개: 화면의 재개 버튼
- 훈련 중 평균 프레임이 50 FPS 아래로 유지되면 성능 경고가 표시됩니다. 다른 탭을 닫거나 그래픽 부담을 줄인 뒤 다시 시도하세요.

## 훈련 모드

- Grid Shot, Micro Flick, Reaction Shot
- Target Switching, Strafe Track, Headshot Only

모든 모드는 3초 카운트다운 후 60초간 진행되며, Easy / Normal / Hard 난이도를 제공합니다. 결과에는 점수, 명중률, 최대 콤보와 모드별 지표가 기록됩니다.

## 감도와 기록

`eDPI = DPI × VALORANT 감도`를 실시간으로 계산하고, 입력한 값에 기본 배율 1.000을 적용합니다. 브라우저는 마우스 DPI를 자동으로 알 수 없으므로 실제 하드웨어 값을 직접 입력해야 합니다. 브라우저 입력 환경은 게임과 다를 수 있으므로 완전히 동일한 감도를 보장하지 않습니다.

감도, 크로스헤어, 최근 기록(모드/난이도별 최대 10개), 개인 최고 기록은 이 브라우저의 `valorant-aim-trainer:data` 로컬 저장소 키에만 보관됩니다. 브라우저 저장소가 막히거나 꽉 찬 경우 화면에 경고가 나타나며, 현재 훈련은 계속할 수 있지만 새로고침하면 변경 사항이 사라질 수 있습니다.

## 면책

이 프로젝트는 비공식 팬 제작 연습 도구입니다. Riot Games 또는 VALORANT와 제휴하거나 승인받지 않았으며, Riot의 로고, 맵, 에이전트, UI, 봇, 사운드 자산을 사용하지 않습니다.
